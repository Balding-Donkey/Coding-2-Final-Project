# Coding 2 Final Project
[Click here to play the latest version](https://balding-donkey.github.io/Coding-2-Final-Project/)
## Description
This is a little platformer game I made from scratch.\
There are three levels. In each level, your goal is to touch the yellow orb. There is no fail state.\
To play, select the "play" option and pick a level. See controls below.
## Notes
- The game is not finished, so I intend to work on it after the project deadline
- The game always runs at a fixed 1120x630 resolution. If this does not fit on your screen well, use your browser's zoom feature to scale the page.
- Mr. Tiller will not be the player character in the final version 😔
- There is an output log that can be viewed by clicking the "Log" button in the bottom right corner. This log exists to get around the fact that the developer console is disabled on school chromebooks.
## Controls
### Menus
- WASD, Arrows, or Left Stick to navigate
- Space, Enter, or gamepad A to select an option
- Escape or gamepad B to go back
- R or gamepad X to reset individual settings to default (unused)
### Ingame
- WASD, Arrows, or Left Stick to move
- Space or gamepad A to jump (Hold the button longer to jump higher)
- Escape or gamepad Start/View to pause/unpause
### Editing
- Z or gamepad Y to toggle edit mode while in a level
- Q/E or LT/RT to change zoom
- Left/Right click to place/delete tiles
- I to place a cat (cannot be removed)
- L to place an ending orb
- K to delete all ending orbs
- J to set the level's start position
- Ctrl + S to save the level
- Ctrl + O to open a level
