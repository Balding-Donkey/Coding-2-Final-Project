# Coding 2 Final Project
Platformer game in javascript

Controls:
- WASD or Arrows to move
- Space to jump (Hold it to jump higher)
- Z to toggle free camera mode
- Q/E to zoom in or out while in free cam
- Click to place blocks (cannot be removed)

[Play in browser](https://balding-donkey.github.io/Coding-2-Final-Project/)
