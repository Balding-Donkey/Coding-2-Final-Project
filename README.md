# Coding 2 Final Project
Platformer game in javascript, made from scratch\
Currently unfinished and using placeholder assets\
The game always runs at a fixed 1120x630 resolution. If this does not fit on your screen well, use your browser's zoom feature to scale the page.

Keyboard controls:
- WASD or Arrows to move and navigate menus
- Space to jump
- Space or Enter to select menu options
- Escape to pause or go back
- R to reset settings to default
- Z to toggle edit mode
- While in edit mode, Q/E to zoom in and out
- While in edit mode, Left/Right Click to place/remove tiles
- While in edit mode, Ctrl + S/O to save/load level files
- I to spawn a cat

Gamepad controls:
- Left stick or Dpad to move and navigate menus
- A (bottom button) to jump or select menu options
- B (right button) to go back
- View/Start to pause and unpause
- X (left button) to reset settings to default
- Y (top button) to toggle edit mode
- While in edit mode, Left/Right Trigger to zoom in and out
- Level editing is not supported for gamepads


[Play in browser](https://balding-donkey.github.io/Coding-2-Final-Project/)
