# Coding 2 Final Project
Platformer game in javascript

Keyboard controls:
- WASD or arrow keys to move
- Space to jump (Hold it to jump higher)
- Z to toggle free camera mode
- Q/E to zoom in or out while in free cam
- I to spawn Glungus
- Click to place blocks, right click to delete blocks
- Ctrl + s to save the current level
- Ctrl + o to load a level

Gamepad controls:
- Left stick to move
- A to jump
- Y to toggle free camera mode
- LT/RT to zoom in or out while in free cam

[Play in browser](https://balding-donkey.github.io/Coding-2-Final-Project/)
